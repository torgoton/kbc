# State of the Game
Where things stand right now

## Authentication
Authentication is mostly working. Users can sign up, log in, log out, and reset their passwords. There are still some edge cases and security improvements to be made, but the core functionality is solid.

## Game Lobby
The game lobby allows users to create and join games.
Players can see a list of available games and their statuses. Some improvements are needed in terms of user experience and real-time updates.

## Gameplay
Basic gameplay mechanics are implemented. Players can take turns, make moves, and the game state updates accordingly.
Live updates on the game view are working
