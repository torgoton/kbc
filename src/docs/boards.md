Each board is a 10x10 grid of hexagons.
The top row is offset to the left.
Rows alternate so that a horizontal line is possible, but a straight vertical one is not.
Boards can be placed in two orientations IRL
  - 4 orientations might be possible for this digital version
Boards have a name according to the location tiles on them

In the base game, there are 5 buildable terrain types and 2 terrains that are not buildable
Each terrain has a representative color
